h1. Java Thread Dump Analyser

This project implements a tool to analyse Java thread dumps.

* Download the "jtda.jar":https://github.com/downloads/mchr3k/javathreaddumpanalyser/jtda.jar
* Run the jar from the command line
** Windows/Linux - "java -jar jtda.jar"
** OSX - "java -jar jtda.jar -XstartOnFirstThread"
* Parse your thread dump text into the input
* Press Analyse
